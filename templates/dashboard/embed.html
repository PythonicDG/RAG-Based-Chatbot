Embed Code

Copy and paste this snippet into your website's HTML to add {{ bot.name }}.

<script src="{{ server_url }}/static/widget.js" data-server="{{ server_url }}" data-bot-id="{{ bot.id }}" data-bot-name="{{ bot.name }}" data-color="{{ bot.primary_color }}" data-welcome="{{ bot.welcome_message }}"> </script>

How to use:

  1. Copy the code snippet above.
  2. Open your website's HTML file.
  3. Paste the snippet just before the closing </body> tag.
  4. Save and refresh your website. The chat bubble will appear automatically!
